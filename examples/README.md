# EVOMEM Examples

This directory contains examples of how to use the EVOMEM library to track interactions, capture code evolution, and export data for SLM training.

## Files
- `basic_capture.py`: How to initialize the memory and record interactions/code changes.
- `validate_and_promote.py`: How to extract golden pairs and validate data before training.
- `export_to_vertex_ai.py`: How to use the optional exporter to simulate sending data to Vertex AI.
