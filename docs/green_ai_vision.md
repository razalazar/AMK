# Green AI Vision

## The Problem
Large Language Models (LLMs) like GPT-4 or Claude 3 provide immense utility but come with a massive environmental cost. Their training and continuous querying demand vast data centers, consuming excessive electricity and water.

## The EVOMEM Solution
**EVOMEM** champions the shift towards **Green AI**. Our thesis is that developers do not need massive generalist models for everyday coding tasks once a specialized domain has been mapped.

By capturing interactions and code evolutions directly from the IDE, EVOMEM generates highly specialized "Golden Datasets". These datasets are used to train Small Language Models (SLMs).

### Benefits of SLMs
1. **Lower Carbon Footprint**: Training requires a fraction of the energy.
2. **Edge Deployment**: Can be run locally on a developer's laptop, eliminating server round-trips.
3. **Data Privacy**: No need to send proprietary code to external APIs.
4. **Faster Inference**: Specialized models return results quicker for their specific domain.
